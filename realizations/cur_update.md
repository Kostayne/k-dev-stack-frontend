## Модели
Пользователь: 

    - Имя, Фамилия
    - Аватарка
    - Имейл
    - Пароль
    - Комментарии к либам
    - Комментарии к проектам

Комментарий:

    - Автор
    - Текст
    - Прикреплен к (проект, либа, коммент)
    - Лайкнувшие пользователи
    - Дата создания

Либа:

    - название
    - описание
    - метки (теги)
    - ссылка на либу для скачивания
    - пример кода
    - вес
    - проекты с этой либой
    - альтернативы этой либы
    - комменты

    - Текущая версия (автоматически)
    - Тип: фреймворк / либа
    - Количество issues (автоматически)
    - Количество скачиваний (автоматически)
    - Последнее обновление (автоматически)
    - Лицензия (Автоматически)

Проект:

    - название
    - описание
    - список  используемых либ (стек)
    - исходники
    - комментарии
    - скриншоты

    - Количество issues (автоматически)
    - Количество звезд (автоматически)
    - Последнее обновление (автоматически)
    - Лицензия (Автоматически)


## Действия
Либа:
```
- Комментировать
- Создать (admin)
- Редактировать (admin)
- Удалить (admin)
```

Проект:
```
- Комментировать
- Создать (admin)
- Редактировать (admin)
- Удалить (admin)
```
